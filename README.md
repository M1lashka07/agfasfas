# GitHub User Finder

Автор: Артём Милосердов

## Описание

GitHub User Finder — GUI-приложение для поиска пользователей GitHub через GitHub REST API.

Приложение позволяет:

- искать пользователей GitHub;
- отображать результаты поиска;
- добавлять пользователей в избранное;
- сохранять избранных пользователей в JSON;
- удалять пользователей из избранного;
- проверять корректность ввода.

Интерфейс реализован с использованием Python и библиотеки tkinter.

---

## Функционал

### Поиск пользователей GitHub

Пользователь вводит имя или логин GitHub-пользователя и получает список найденных аккаунтов.
